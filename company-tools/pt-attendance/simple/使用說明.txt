  掃 QR Code 直接打卡！

🎯 兩種打卡方式：

方式一：掃描 QR Code（最方便！）⭐
1. 用手機相機掃描 QR Code
2. 自動開啟打卡頁面並顯示員工資訊
3. 選擇「上班打卡」或「下班打卡」
4. 完成！

方式二：手動輸入編號
1. 開啟 打卡.html
2. 輸入員工編號（例如：E001）
3. 按「確認」
4. 選擇「上班打卡」或「下班打卡」
5. 完成！

==========================================

📱 QR Code 超方便！

現在的 QR Code 掃描後會：
✅ 自動開啟打卡頁面
✅ 自動帶入員工資訊
✅ 直接顯示上班/下班按鈕
✅ 不用輸入任何東西

就像掃描 LINE 加好友一樣簡單！

使用步驟：
1. 管理員在 index.html 新增員工
2. 下載 QR Code 圖片給員工
3. 員工把 QR Code 存在手機
4. 要打卡時，用手機相機掃 QR Code
5. 自動開啟打卡頁面
6. 點「上班打卡」或「下班打卡」
7. 完成！

==========================================

📁 檔案說明：

1. index.html
   - 管理員用
   - 新增/刪除員工
   - 生成 QR Code（掃了會直接開打卡頁面）
   - 查看所有打卡記錄

2. 打卡.html ⭐ 員工打卡專用
   - 可以手動輸入編號
   - 也可以掃 QR Code 自動開啟
   - 會顯示今日已打卡記錄

3. 使用說明.txt
   - 就是這個檔案

💡 QR Code 使用技巧：

1. 列印出來
   - 把 QR Code 列印成小卡片
   - 員工隨身攜帶
   - 打卡時拿出來掃

2. 存在手機
   - 把 QR Code 圖片存在手機相簿
   - 或設為桌布
   - 要打卡時開啟相簿給別人掃

3. 貼在工作證
   - 把 QR Code 列印後護貝
   - 貼在工作證背面
   - 方便又專業

4. LINE 個人頭像
   - 把 QR Code 設為 LINE 頭像
   - 隨時可以請別人掃
   - （開玩笑的，但也可以）

==========================================

💾 資料說明：

- 所有資料存在瀏覽器 localStorage
- 不會消失（除非清除瀏覽器資料）
- 只在這台電腦的這個瀏覽器有效
- 建議定期截圖備份記錄

注意：
- 掃 QR Code 是開啟網頁，不是儲存資料
- 所以不同手機掃同一個 QR Code 都會看到同樣的資料
- 資料還是存在電腦的瀏覽器裡

⚠️ 注意事項：

1. 檔案位置很重要
   - index.html 和 打卡.html 要在同一個資料夾
   - 否則 QR Code 掃描後找不到打卡頁面

2. 本地檔案的限制
   - 如果是雙擊開啟檔案（file:///...）
   - QR Code 可能無法正常運作
   - 建議：
     a) 放到網頁伺服器（最佳）
     b) 或使用 Python 簡易伺服器（見下方）

3. 啟動簡易伺服器（推薦）
   - 開啟命令提示字元
   - 切換到專案資料夾
   - 輸入：python -m http.server 8000
   - 用瀏覽器開啟：http://localhost:8000
   - 這樣 QR Code 就能正常運作

4. 手機要能連到電腦
   - 如果用 localhost，只有電腦能用
   - 要讓手機掃 QR Code 能用，需要：
     a) 把檔案放到網頁空間（GitHub Pages 等）
     b) 或用區域網路 IP（例如 192.168.1.100:8000）

❓ 常見問題：

Q: 掃了 QR Code 沒反應？
A: 檢查：
   1. 檔案是否在網頁伺服器上
   2. 手機能否連到這個網址
   3. 試試用手動輸入方式

Q: QR Code 太小掃不到？
A: 在 index.html 下載 QR Code 後
   可以用圖片檢視器放大再掃

Q: 可以一個 QR Code 給多人用嗎？
A: 不行，每個員工要有自己的 QR Code
   因為 QR Code 裡包含員工編號

Q: 打錯了怎麼辦？
A: 到 index.html 的「記錄查詢」
   可以看到所有記錄
   （目前無法刪除單筆記錄）

Q: 需要網路嗎？
A: 不需要網路！
   但手機要能連到檔案所在的網址
   （可以是區域網路）

==========================================

🎉 推薦使用方式：

【最簡單：區域網路】
1. 在打卡地點的電腦啟動簡易伺服器
2. 記下電腦的區域網路 IP
   （例如：192.168.1.100）
3. 生成 QR Code 時，網址會自動包含這個 IP
4. 員工用手機掃 QR Code
5. 手機會連到電腦開啟打卡頁面
6. 完成打卡

【最穩定：網頁空間】
1. 把 index.html 和 打卡.html 上傳到網頁空間
   （GitHub Pages、Google Drive 等）
2. 生成 QR Code
3. QR Code 會連到網路上的網址
4. 員工隨時隨地都能掃描打卡

【最簡易：固定電腦】
1. 在固定電腦開啟 打卡.html
2. 員工來打卡時，手動輸入編號
3. 不用 QR Code 也很方便
