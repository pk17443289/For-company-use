# 人員管理表 - 戰術調度系統

專為軍中或大型組織設計的人員管理與任務調度系統，採用 **Cyberpunk 電競風格**，支援快速查找空閒人員、拖拉分配任務、即時狀態追蹤等功能。

## 核心特色

### 🎮 Cyberpunk 電競風格
- 黑色背景 + 黃色主色調
- 霓虹發光效果
- 動態網格背景
- Glitch 動畫效果
- Consolas 等寬字體

### 🚀 專為大量人員設計
- **卡片式網格視圖**：即使有 100 人也能一目了然
- **智能狀態標示**：綠色發光 = 空閒、紅色發光 = 忙碌、橘色發光 = 部分空閒
- **快速篩選搜尋**：秒找到需要的人員
- **時段切換**：即時查看不同時段的人力狀態

### ⚡ 核心功能

1. **時段選擇器**
   - 快速切換：現在、上午、下午、晚上
   - 自訂時段：精確設定任意時間範圍
   - 即時更新人員狀態

2. **智能篩選**
   - 狀態篩選：全部/空閒/忙碌/部分空閒
   - 位階篩選：高階/中高階/中階/基層/新進
   - 即時搜尋：輸入姓名快速定位

3. **視覺化人員狀態**
   - 霓虹邊框：根據狀態發光（綠/紅/橘）
   - 位階標示：LV 1-10 等級系統
   - 一鍵查看詳情

4. **拖拉分配任務**
   - 從任務池拖拉任務卡片
   - 拖到人員卡片上即可分配
   - 自動記錄歷史

5. **任務管理**
   - 三種類型：日常（綠）、重要（紅）、臨時（橘）
   - 時段設定：以小時為單位（0-23）
   - 快速篩選：全部/日常/重要/臨時/未分配

6. **統計儀表板**
   - 人員統計：總人數、各位階人數
   - 任務統計：總任務、已分配、未分配
   - 歷史記錄：最近 100 筆操作記錄

## 使用場景

### 情境 1：早晨調度
```
1. 開啟系統，自動顯示當前時段
2. 查看頂部統計：5 人空閒、3 人忙碌
3. 點擊「空閒」篩選按鈕，立即看到 5 個綠色發光的卡片
4. 選擇合適人員，拖拉任務卡片進行分配
```

### 情境 2：臨時任務出現
```
1. 點擊「+ 新增任務」
2. 填寫任務資訊：名稱、類型（臨時）、時段（14:00-16:00）
3. 系統自動顯示在任務池
4. 切換時段到下午（13-17）
5. 點擊「空閒」篩選，找出該時段可用人員
6. 拖拉任務到選定人員，完成分配
```

### 情境 3：查詢高階人員狀態
```
1. 在位階篩選選擇「高階 (9-10)」
2. 系統立即顯示所有高階人員
3. 查看其狀態：綠色發光代表有空
4. 點擊卡片查看詳細任務列表
```

### 情境 4：檢查未分配任務
```
1. 點擊任務池的「未分配」標籤
2. 查看所有尚未分配的任務
3. 選擇合適時段
4. 篩選出該時段的空閒人員
5. 逐一拖拉分配任務
```

## 操作指南

### 時段切換
- **現在**：顯示當前小時的人員狀態
- **上午**：08:00-12:00
- **下午**：13:00-17:00
- **晚上**：18:00-22:00
- **自訂**：設定任意時段範圍

### 狀態說明
- **🟢 空閒（綠色發光）**：該時段完全沒有任務
- **🔴 忙碌（紅色發光）**：該時段完全被任務占滿
- **🟠 部分空閒（橘色發光）**：該時段有部分時間有空

### 位階系統
- **9-10 級**：高階（深藍色標示）
- **7-8 級**：中高階（藍綠色標示）
- **5-6 級**：中階（綠色標示）
- **3-4 級**：基層（黃色標示）
- **1-2 級**：新進（淺藍色標示）

### 任務類型
- **日常（綠色）**：例行工作，如巡視、檢查
- **重要（紅色）**：重大任務，如會議、訓練
- **臨時（橘色）**：臨時支援、突發狀況

### 拖拉分配
1. 從左側任務池選擇任務卡片
2. 按住滑鼠左鍵開始拖曳
3. 拖到目標人員卡片上
4. 放開滑鼠完成分配
5. 系統自動更新狀態並記錄歷史

### 查看詳情
- **點擊人員卡片**：在右側面板顯示該人員的詳細資訊和任務列表
- **雙擊任務卡片**：開啟編輯對話框
- **點擊統計報表**：查看完整的統計數據和歷史記錄

- 資料存在 `localStorage`
- 快速、無需網路
- 僅限單一瀏覽器

### Google Sheets 模式（選用）
- 資料存在 Google 試算表
- 可跨裝置同步
- 支援多人協作
- 設定方法請參考 [SHEETS_SETUP.md](SHEETS_SETUP.md)

## 常見問題

**Q: 可以管理多少人員？**
A: 理論上無上限，實測 200 人以上仍流暢運作。卡片式網格設計能有效組織大量人員資訊。

**Q: 如何快速找到空閒人員？**
A: 點擊「空閒」篩選按鈕，系統會只顯示當前時段完全空閒的人員（綠色發光）。

**Q: 可以跨裝置使用嗎？**
A: 本地模式僅限單一瀏覽器。如需跨裝置，請設定 Google Sheets 同步（參考 SHEETS_SETUP.md）。

**Q: 任務時段可以跨夜嗎？**
A: 目前僅支援當日 0-23 時，不支援跨日任務。

**Q: 如何備份資料？**
A: 方法 1：使用 Google Sheets 同步。方法 2：開啟瀏覽器開發者工具，從 localStorage 匯出資料。

**Q: 可以列印或匯出排班表嗎？**
A: 目前僅支援螢幕檢視。未來版本將加入匯出功能。
